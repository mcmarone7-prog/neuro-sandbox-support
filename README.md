# Neuro SandBox

Neuro SandBox is an interactive neural simulation sandbox for exploring firing dynamics, synaptic decay, and network behavior in real time.

Designed for students, researchers, and anyone curious about computational neuroscience.

## Features
- Build and visualize neural networks
- Adjust firing rates and synaptic parameters
- Observe emergent dynamics live
- Runs fully on-device
